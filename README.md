## Swift Property Showcase

This is a **LuxeEstate**-style luxury real estate property showcase built with modern React tooling.

### Tech stack

- **Vite**
- **TypeScript**
- **React**
- **shadcn-ui**
- **Tailwind CSS**

### Getting started

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

### Building for production

```sh
npm run build
```
